# تغییرات نسخه‌ها

اینجا می‌بینی هر نسخه چی جدید آورده — ساده و خلاصه ✨

## [1.3.0] — 2026-07-24

منوی + در کامپوزر و کنترل دقیق‌تر MCP برای هر گفتگو 🔌

### چی جدیده؟

- ➕ **منوی افزودن در کامپوزر** — با دکمهٔ + فایل، سرور MCP، مهارت یا متخصص را مستقیم به پیام اضافه کنید؛ پنل تمام‌عرض زیر/بالای کادر پیام باز می‌شود
- 🔌 **MCP برای هر گفتگو** — از همان منو مشخص کنید کدام سرورهای فضای کاری در این چت فعال باشند؛ انتخاب ذخیره می‌شود
- ⚙️ **صفحهٔ تنظیمات MCP** — مدیریت سرورهای هر فضای کاری از تنظیمات → سرورهای MCP
- 🎤 **میکروفن تمیزتر** — انتخاب دستگاه و تست سطح صدا در تنظیمات گفتار جمع‌وجورتر شده
- ⏱️ **رونویسی تا دو ساعت** — سقف مدت فایل صوتی برای رونویسی به دو ساعت افزایش یافته است
- 🧭 **تور و راهنما** — مرحلهٔ MCP به تور شروع اضافه شده و راهنما توضیح منوی + و سرورهای MCP را دارد

## [1.2.0] — 2026-07-21

نیمروز حالا گفت‌وگو و رونویسی فارسی را طبیعی‌تر و خصوصی‌تر می‌کند 🎙️

### چی جدیده؟

- 🎤 **ورودی صوتی در چت** — با میکروفن صحبت کن تا صدایت مستقیماً به متن پیام در کامپوزر تبدیل شود
- 🗣️ **مدل‌های فارسی Shenava** — مدل‌های ASR شنوا را دانلود و روی دستگاه اجرا کن؛ رونویسی بدون فرستادن صدای تو به سرویس خارجی انجام می‌شود
- 📝 **صفحهٔ رونویسی اختصاصی** — فایل صوتی یا صدای میکروفن را به متن فارسی تبدیل کن، خروجی بگیر و متن خام و نسخهٔ ویرایش‌شده را کنار هم نگه دار
- ✨ **اصلاح با هوش مصنوعی** — بعد از رونویسی، متن را با مدل دلخواهت برای نشانه‌گذاری، فاصله‌گذاری و خوانایی بهتر اصلاح کن
- 💬 **حالت چت** — یک گفت‌وگوی سبک و متمرکز، بدون ابزار و بدون زمینهٔ workspace؛ فقط یک چت‌بات برای پرسش و پاسخ روزمره
- ⚙️ **تنظیمات گفتار** — دانلود، انتخاب و مدیریت مدل شنوا در onboarding و تنظیمات اپ

## [1.1.0] — 2026-07-20

نیمروز حالا برای کارهای چندمرحله‌ای، برنامه‌ریزی و ویرایش پروژه خیلی روان‌تر و قابل‌کنترل‌تره 🚀

### چی جدیده؟

- 🧭 **حالت Plan** — قبل از اجرای کارهای بزرگ، برنامه بساز، سؤال‌های لازم رو جواب بده و روند پیشرفت رو در پنل اختصاصی دنبال کن
- 🤝 **عامل‌های فرعی** — کارهای مستقل می‌تونن بین subagentها تقسیم بشن و نتیجه‌ها به گفت‌وگوی اصلی برگردن
- 🛡️ **اجرای امن‌تر ابزارها** — سیاست‌های دسترسی، اعتبارسنجی مسیرها و کنترل shell دقیق‌تر شده
- ⌨️ **تأیید دستورهای ترمینال** — دستورها با ظاهر خواناتر ترمینال و دکمه‌های جمع‌وجور نمایش داده می‌شن؛ پیام جدید هم پشت کارت تأیید گیر نمی‌کنه
- ⏹️ **وضعیت درست توقف و لغو** — ابزارهای لغوشده، ردشده یا متوقف‌شده دیگه اشتباهی با spinner و وضعیت «در حال اجرا» باقی نمی‌مونن
- 🗂️ **فایل‌اکسپلورر تازه** — نمای تمام‌عرض شبیه ادیتور، فهرست فایل‌های تغییرکرده و آمار خطوط اضافه/حذف‌شده در پایین پنل
- ✍️ **ویرایش مستقیم فایل** — فایل‌های متنی بدون جابه‌جایی بین preview و edit باز می‌شن؛ دکمه ذخیره فقط وقتی تغییری وجود داشته باشه ظاهر می‌شه
- 🎨 **syntax highlighting** — فایل‌های کد، JSON و Markdown هنگام ویرایش با پشتیبانی تم روشن و تاریک رنگ‌آمیزی می‌شن

## [1.0.0] — 2026-07-17

نیمروز دیگه فقط چت نیست — حالا یک **workspace agent** واقعیه 🚀

### چی جدیده؟

- 🗂️ **فضای کاری** — پوشه پروژه‌ات رو وصل کن؛ دستیار داخلش فایل می‌خونه، جستجو می‌کنه، shell می‌زنه و artifact می‌سازه
- 🤖 **Codex** — با اشتراک ChatGPT وصل شو و از مدل‌های Codex استفاده کن (جدا از فایل‌های workspace)
- 📌 **پنل کناری** — فایل‌ها، artifactها، taskها و فعالیت فضای کاری، همه یک‌جا
- 🔗 **تایم‌لاین ابزار** — ببین دستیار قدم‌به‌قدم چیکار کرده (برای کارهای طولانی جمع‌وجور می‌شه)
- ✍️ **کامپوزر هوشمند** — workspace picker، context chip و `@` برای فایل و artifact
- 🎨 **ظاهر** — تم رنگی، فونت سیستم، accent نیمروز، و onboarding اولین بار
- ✨ **تازه‌ها** — بعد از update یه popup کوتاه می‌بینی؛ تاریخچه کامل توی تنظیمات → تغییرات نسخه‌ها
- 🏠 **پروژه → workspace** — همه‌چیز به مدل workspace منتقل شد (با یک Home workspace)
